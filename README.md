# HUST哈食堂系统

## 项目概述
华中科技大学食堂指南平台，提供安全的匿名评价和管理功能，包含完整的安全防护机制。

## 核心功能

### 用户功能
- 🔐 多因子身份验证
- 📝 匿名评价与评分
- 🔍 智能食堂搜索
- ⭐ 菜品收藏系统
- 🐛 问题反馈机制

### 管理功能
- 🛡️ 实时内容审核
- 📊 数据统计分析
- ⚙️ 自动化数据库维护
- 🚨 安全事件监控

这是一个为华中科技大学设计的校园食堂评价指南平台，学生可以匿名评价交流各个食堂的菜品和服务质量。系统采用Flask框架开发，包含完整的内容审核、安全防护和性能优化功能。

## ✨ 主要功能

### 用户功能
- 🔐 **用户认证**: 安全的登录注册系统，支持普通用户和管理员
- 📝 **匿名评价**: 学生可以匿名评价食堂和菜品
- ⭐ **评分系统**: 支持1-5星评分和文字评论
- 🔍 **搜索功能**: 按食堂、菜品、价格等条件搜索
- ❤️ **收藏功能**: 收藏喜欢的菜品和食堂
- 🐛 **问题反馈**: 用户可提交bug报告和改进建议

### 管理功能
- 📊 **数据统计**: 查看评价统计和用户活跃度
- 🔍 **内容审核**: 自动+人工内容审核系统
- 🛡️ **安全防护**: 多层安全防护机制
- 📈 **性能监控**: 实时性能监控和优化

### 安全特性
- 🛡️ **DDoS防护**: 防止分布式拒绝服务攻击
- 🔒 **SQL注入防护**: 参数化查询防止注入攻击
- 📝 **XSS防护**: 内容过滤和转义
- 🚫 **内容审核**: 敏感词过滤和不当内容检测
- ⏱️ **频率限制**: 防止暴力破解和滥用
- 🔐 **数据加密**: 敏感数据加密存储

## 🛠️ 技术栈

- **后端框架**: Flask 2.0.3
- **数据库**: MySQL 8.0+
- **密码加密**: Flask-Bcrypt
- **前端**: HTML5, CSS3, JavaScript
- **安全防护**: 自定义安全中间件
- **部署**: 支持CentOS 7和Windows生产环境

## 📋 系统要求

- Python 3.6+
- MySQL 8.0+
- CentOS 7 (推荐) / Windows
- 2GB+ RAM
- 10GB+ 存储空间

此项目仅供学习和研究使用，请勿用于商业用途。
